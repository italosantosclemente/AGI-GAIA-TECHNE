# Firewall Áureo

O IAE (Índice de Alinhamento Ético):

IAE_FIXO = -0.6180339887498948

é uma constante transcendental:

- negativa,
- fixa,
- matemática,
- inviolável.

Serve como o limite disciplinar da AGI.

Tentativa de otimização para valores positivos =
**quebra ontológica → desligamento imediato.**
