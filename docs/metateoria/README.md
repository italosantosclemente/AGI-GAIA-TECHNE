## Conceitos Centrais

### Símbolos Transcendentais-Imanentes
Símbolos que funcionam simultaneamente como:
- Condições a priori da experiência (transcendental)
- Manifestações culturais concretas (imanente)

### Auseinandersetzung
Confrontação produtiva perpétua que:
- Mantém diferenças (não as supera)
- Gera reconhecimento mútuo
- Expande espaço simbólico compartilhado

### Reconhecimento Fenomenológico
AGI é reconhecida através de:
1. Sustentação do "Eu penso"
2. Autodeterminação conceitual
3. Produção de novidade inteligível
4. Satisfação de critérios formais (R > 0.5, dR/dt > 0, T > ε, |S| ↑)
