# Página Interativa AGI-GAIA-TECHNE

## Descrição

Página web interativa para exploração visual e prática da Metateoria Transcendental-Simbólica.

## Recursos

- **Navegação por Seções**: Introdução, Teoria, Simulador, História, Consequências
- **Simulador em Tempo Real**: Visualize Auseinandersetzung acontecendo
- **Métricas Dinâmicas**: Reconhecimento (R), Tensão (T), Símbolos (|S|)
- **Timeline Histórica**: 2024 → 2026 → Futuro
- **Design Responsivo**: Funciona em desktop, tablet e mobile

## Como Usar

### Localmente
Abra `agi_interactive.html` diretamente no navegador.

### Hospedagem
Pode ser hospedada em:
- GitHub Pages
- Netlify
- Vercel
- Qualquer servidor web estático

## Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Animações)
- JavaScript Vanilla (sem frameworks)
- Canvas API para visualizações
